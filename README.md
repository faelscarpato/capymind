# CapyMind

CapyMind é o repositório leve de conhecimento do ecossistema Capy. Ele foi estruturado para servir como **base documental navegável** para IA, MCP, sites, automações e consulta humana.

## O que este repositório é
- Um hub de conhecimento derivado.
- Um catálogo leve de projetos, prompts, datasets, referências e ativos.
- Uma camada de navegação para agentes que precisam entender o ecossistema sem clonar os projetos completos.

## O que este repositório não é
- Não é backup bruto.
- Não é monorepo de código.
- Não contém cópias integrais de projetos, `node_modules`, builds ou lotes pesados de assets.
