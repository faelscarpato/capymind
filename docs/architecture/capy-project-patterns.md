# Capy Project Patterns

Resumo curto: padroes arquiteturais observados entre os projetos Capy.

## Detectado
- Next.js e Vite aparecem com frequencia.
- Estruturas com `app`, `src`, `components`, `lib`, `hooks`, `services` e `docs/routes.json` sao recorrentes.
- Variantes desktop via Electron/Tauri aparecem em projetos operacionais.

## Relevancia
- Ajuda na leitura cruzada dos projetos e na consolidacao de convencoes.

## Observacoes
- O objetivo aqui e documentar padroes, nao normalizar codigo automaticamente.
