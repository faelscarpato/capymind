# Important Files

- `package.json`: Defines project name, scripts, and dependencies.
- `README.md`: Summarizes intent and usage.
- `vite.config.ts`: Shows the Vite build pipeline.
- `tsconfig.json`: Defines the TypeScript baseline.
- `tailwind.config.js`: Captures the visual token and utility setup.
- `src/main.tsx`: Likely UI entry point.
- `src/App.tsx`: Likely root component.
- `src/app`: Key directory for architecture reading.
- `src/components`: Key directory for architecture reading.
- `supabase`: Key directory for architecture reading.
- `functions`: Key directory for architecture reading.
- `public`: Key directory for architecture reading.
