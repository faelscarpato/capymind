# capyide

Resumo curto: # CapyIDE CapyIDE é um editor de código online assistido por IA focado na prototipação rápida de páginas web. A aplicação utiliza o Monaco Editor (com fallback para um `<textarea>` simples) para edição, oferece pré-visualização em tempo real do HTML e integra-se ao modelo Google Gemini para gerar ou editar código a partir de descrições em linguagem natural. ## Principais recursos - **Geração guiada por IA** – descreva o projeto desejado em português e receba um HTML completo com CSS/JS embutidos. - **Edição iterativa** – utilize o chat para solicitar ajustes no código atual. O CapyIDE decide automaticamente quando deve gerar uma nova versão ou editar a existente. - **Pré-visualização instantânea** – o iframe de preview é atualizado a cada alteração no código, permitindo validar rapidamente o resultado.
