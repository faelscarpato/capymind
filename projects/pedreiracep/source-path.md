# Source Path — PedreiraCep

## Resumo

Este documento registra a origem oficial do projeto PedreiraCep e reforça que o CapyMind mantém apenas documentação derivada e conhecimento estruturado sobre o projeto, não uma cópia completa da implementação.

## Origem principal

- Repositório original: `https://github.com/faelscarpato/pedreiracep`
- Publicação web: `https://faelscarpato.github.io/pedreiracep/`

## Papel do repositório original

O repositório original é a fonte principal para:

- código-fonte
- arquivos de interface
- scripts de comportamento
- dataset utilizado no projeto
- documentação de implementação
- histórico técnico do projeto

## Papel do CapyMind

No CapyMind, o PedreiraCep é representado como conhecimento derivado, incluindo:

- resumo do projeto
- propósito
- stack percebida
- arquitetura
- notas sobre dados
- mapa dos arquivos principais
- observações de integração
- documentação de contexto

## Regra operacional

O código completo do projeto **não foi copiado** para o CapyMind.

O CapyMind registra:

- entendimento semântico
- navegação documental
- estrutura resumida
- pontos relevantes para IA e consulta humana

## Justificativa

Essa separação é importante para manter:

- o `pedreiracep` como repositório de implementação
- o `capymind` como repositório de conhecimento
- clareza de papéis entre execução e documentação
- leveza estrutural do CapyMind

## Fonte de verdade

### Para código e execução
- repositório original `pedreiracep`

### Para conhecimento derivado e navegação entre projetos
- `capymind`

## Observações finais

Qualquer atualização estrutural relevante do PedreiraCep deve ser refletida posteriormente na documentação do CapyMind, mantendo a rastreabilidade entre o repositório de implementação e o repositório de conhecimento.
